# MExpense

MExpense is a simple mobile expense tracker app built with Flutter for the App Development Boot Camp May 2026 assignment.

## Features

- Home dashboard
- Total monthly expense summary
- Daily average and budget left
- Add new expense
- Category selection
- Date and note fields
- Expense history
- Delete expense

## UI Design

The UI was created using Google Stitch and converted into a Figma file.
