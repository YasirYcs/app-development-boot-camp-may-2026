# MExpense - App Development Bootcamp May 2026

MExpense is a simple expense tracking app built with Flutter for the APP Development Bootcamp.

The goal of this project was to create a clean and practical mobile application within a short time while keeping the architecture simple and easy to maintain.

The app allows users to add expenses, track spending, and view basic financial summaries with a modern and minimal UI.

---

## Features

- Add new expenses
- Delete expenses
- View recent transactions
- Monthly expense overview
- Expense categories
- Simple statistics section
- Local data storage using Hive
- Clean responsive UI
- Animated splash screen

---

## Tech Stack

- Flutter
- Provider
- Hive
- Material 3

